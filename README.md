# React-chap-app

A chat web application powered which allow user to join a room and chat.

## Description

A learning project that I build to learn Socket io, Express js, and tailwind css with React.

This app is using Vite for frontend development and Node js for backend. It is not a production app. To run the app, using these following command:

** Run each commands in separate terminal

`cd frontend npm run dev`

`cd backend npm run dev`
